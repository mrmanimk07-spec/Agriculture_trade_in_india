# Agriculture_trade_analysis_in_india

# 🌾 India Agricultural Trade Analysis (Import & Export)

## 📌 Project Overview

This project focuses on analyzing India’s agricultural import and export data using data cleaning, transformation, and exploratory data analysis (EDA). The dataset initially contained missing values, inconsistent formats, and mixed data types, which were handled to generate meaningful business insights.

---

## 🎯 Objective

* Clean and preprocess messy data
* Handle missing values and data inconsistencies
* Perform exploratory data analysis (EDA)
* Identify key trade patterns and trends
* Generate actionable business insights

---

## ⚠️ Problem Statement

Agricultural trade data is often unstructured and contains missing values and inconsistencies, making it difficult to analyze. This project aims to transform raw data into a clean format and extract meaningful insights for decision-making.

---

## 🧹 Data Cleaning Steps

* Handled missing values (median & mode imputation)
* Removed duplicates
* Converted mixed data types
* Treated outliers using IQR method
* Standardized dataset

---

## 🔄 Data Transformation

* Created new features:

  * Trade Balance
  * Value per Tonne
* Categorized data using `pd.cut()`
* Converted columns into appropriate data types

---

## 📊 Exploratory Data Analysis (EDA)

### ✔ Univariate Analysis

* Distribution of export/import values
* Category-wise distribution

### ✔ Bivariate Analysis

* Export vs Import relationship
* Category vs Export performance
* Correlation analysis

### ✔ Multivariate Analysis

* Country & category analysis
* Year-wise trends
* Pairplot for multiple relationships

---

## 📈 Key Insights

* Exports and imports are positively related
* Few high-value transactions drive major revenue
* Trade is dominated by specific product categories
* Top countries contribute most of the trade
* Tariff and exchange rate have limited impact
* Organic products show growth potential

---

## 💡 Business Recommendations

* Focus on high-performing products
* Strengthen relationships with key countries
* Expand into new markets
* Promote organic products
* Use data-driven strategies for decision-making
